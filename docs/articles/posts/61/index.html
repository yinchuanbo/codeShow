使用表格显示属性实现布局

Sep 25, 2023 · Yin灏
  • HTML
<ul>
  <li>HOME</li>
  <li>SPACESHIPS</li>
  <li>PLANETS</li>
  <li>STRAS</li>
</ul>
  • CSS
ul {
  width: 100%;
  display: table;
  table-layout: fixed;
}

li {
  width: 25%;
  height: 50px;
  background-color: red;
  list-style: none;
  display: table-cell;
  vertical-align: middle;
  text-align: center;
}
li:nth-child(2) {
  background-color: aqua;
}
li:nth-child(3) {
  background-color: aquamarine;
}
li:nth-child(4) {
  background-color: blue;
}
